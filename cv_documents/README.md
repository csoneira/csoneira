# CV Documents

This folder is the source of truth for certificates, letters, and supporting evidence used in the CV and website.
See `cv_documents/cv_list.md` for the working CV list and `cv_documents/document_index.md` for document mapping.

## Structure

- `cv_documents/PUBLIC`: sanitized documents that can be referenced from the site
- `cv_documents/PRIVATE`: private or administrative files (not published)
- `cv_documents/NOT_ESSENTIAL`: legacy material or archives not needed for current work
